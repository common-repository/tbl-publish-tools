== Description ==

Two Bright Lights Publishing Tool allows you to automatically mark your submission as published upon publishing your post in WordPress. No more copying and pasting the feature link into Two Bright Lights, the information will be transferred over to the system and will be included in the vendor notifications.

Note:
*   This plugin depends on twobrightlights.com's API(https://twobrightlights.com/apis/1.0/updateSubmissionStatus). Please make sure your site is avaible to visit internet web sites before using the plugin.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tbl_publish_tools` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
